# تطبيق متابعة الصحة النفسية 🧠💚 (Mental Health Monitoring App)

**تطبيق لدعم ومتابعة الصحة النفسية. يتيح التطبيق كتابة اليوميات، وتتبع المزاج من خلال تحليل النصوص، بالإضافة إلى إجراء مقاييس التقييم الذاتية.**

---

## الميزات الأساسية 🌟
- **اليوميات الذكية:** مساحة لكتابة يومياتك مع تحليل ذكي لنصك لمعرفة حالتك المزاجية.
- **تتبع المزاج:** رسوم بيانية تفاعلية لحالتك المعنوية على مدار الأيام الماضية.
- **الاختبارات النفسية:** مقاييس علمية (PHQ-9 للاكتئاب و GAD-7 للقلق) مع حفظ لتاريخ وتطور حالتك.
- **التحدث مع AI:** واجهة محادثة ذكية مبدئية لتقديم الدعم الفوري (تصميم أولي).

---

## متطلبات التشغيل ⚙️
- Flutter SDK (الإصدار الأحدث المستقر)
- Android Studio / Xcode

---

## خطوات التشغيل 🚀

1. قم بعمل استنساخ للمستودع (Clone) إلى جهازك:
   ```bash
   git clone [رابط المستودع]
   ```
2. ادخل إلى مجلد المشروع:
   ```bash
   cd mental_health_app
   ```
3. قم بتثبيت الحزم البرمجية المعتمدة:
   ```bash
   flutter pub get
   ```
4. شغل التطبيق على المحاكي أو الجهاز المتصل:
   ```bash
   flutter run
   ```



---

## هيكلية المجلدات الأساسية 📁
- `lib/main.dart` : نقطة البداية للمشروع.
- `lib/database/` : يحتوي على إعدادات قاعدة البيانات المحلية (SQLite) وتوليد البيانات التجريبية.
- `lib/models/` : فئات البيانات الأساسية (User, JournalEntry, NLPResult, Assessment).
- `lib/providers/` : إدارة الحالة والدوال المركزية (مثل التسجيل وتسجيل الدخول).
- `lib/screens/` : ملفات الواجهات لجميع الشاشات في التطبيق (Home, Login, Journal, MoodTracker...).
- `lib/widgets/` : مكونات واجهة المستخدم القابلة لإعادة الاستخدام.
